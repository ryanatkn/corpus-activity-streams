Link

category 'vocab.core'
extends Entity
extended_by Mention
properties hrefrelhreflangheightwidth
A Link is an indirect, qualified reference to a resource identified by a URL. The fundamental model for links is established by [ RFC5988 ]. Many of the properties defined by the Activity Vocabulary allow values that are either instances of Object or Link . When a Link is used, it establishes a RFC5988qualified relation connecting the subject (the containing object) to the resource identified by the href . Properties of the Link are properties of the reference as opposed to properties of the resource.
{
"@context" : " https://www.w3.org/ns/activitystreams " ,
" type " : " Link " ,
" href " : " http://example.org/abc " ,
" hreflang " : "en" ,
" mediaType " : "text/html" ,
" name " : "An example link" ,
}

Activity

category 'vocab.core'
extends Object
extended_by AcceptAddAnnounceCreateDeleteDislikeFlagFollowIgnoreIntransitiveActivityJoinLeaveLikeListenMoveOfferReadRejectRemoveUndoUpdateView
properties actorobjecttargetoriginresultinstrument
An Activity is a subtype of Object that describes some form of action that may happen, is currently happening, or has already happened. The Activity type itself serves as an abstract base type for all types of activities. It is important to note that the Activity type itself does not carry any specific semantics about the kind of action being taken.
{
"@context" : " https://www.w3.org/ns/activitystreams " ,
" type " : " Activity " ,
" summary " : "Sally did something to a note" ,
" actor " : {
" type " : " Person " ,
" name " : "Sally" ,
} ,
" object " : {
" type " : " Note " ,
" name " : "A Note" ,
} ,
}
